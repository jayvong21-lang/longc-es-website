# 厦门龙溪传芳企业服务有限公司官网

一个专业的企业服务官方网站，展示厦门龙溪传芳企业服务有限公司的核心业务、项目案例、团队介绍和联系方式。

## 项目简介

这是一个静态网站项目，用于展示厦门龙溪传芳企业服务有限公司的服务内容和企业形象。网站采用现代设计风格，响应式布局，适合在各类设备上浏览。

## 技术特点

- **HTML5 + CSS3 + JavaScript**: 纯前端技术栈，无需后端服务器
- **响应式设计**: 适配桌面、平板和手机设备
- **现代UI设计**: 采用渐变背景、卡片式布局和动画效果
- **SEO友好**: 结构清晰的HTML语义化标签
- **快速加载**: 压缩图片资源，优化性能

## 部署方式

### 本地部署
1. 克隆项目到本地
2. 直接在浏览器打开 `index.html` 文件即可预览

### GitHub Pages部署
1. 在GitHub创建仓库并上传项目文件
2. 在仓库设置中启用GitHub Pages
3. 选择 `main` 分支作为部署源
4. 网站将自动部署到 `https://<username>.github.io/<repository-name>`

### 自定义域名部署
1. 在GitHub Pages设置中添加自定义域名
2. 在域名DNS设置中添加CNAME记录指向GitHub Pages
3. 等待DNS生效后即可访问
