# 龙溪企服官方网站

龙溪企服官方网站是一个灰色系商务简约风格的网站，采用一级菜单结构，包含首页、企业概况、产品服务、资讯中心、入驻企业、立即入驻等6个核心页面。

## 网站特点

1. **灰色系商务简约风格**：采用灰色系配色方案（背景色#f8f9fa浅灰色、主色#6c757d中灰色、强调色#495057深灰色、文字色#212529深黑色、链接色#007bff蓝色点缀）
2. **一级菜单结构**：简洁明了的导航栏，包含6个核心页面
3. **移动端优先设计**：响应式设计，适配各种设备屏幕
4. **SEO优化**：完整的meta标签和关键词优化
5. **无动画效果**：简洁的设计风格，不包含动画效果

## 网站内容

### 企业简介
龙溪企服"LongC Enterprise Services"，总部位于福建厦门，公司致力于为中小微企业提供专业化的、一站式的企业服务解决方案。为创业者提供涵盖创立、孵化、发展三个阶段，包括办公场所、财务管理、人力资源服务、营销策划、政策咨询等五个方面的企业发展全要素服务。

### 经营范围
1. **办公场所**：根据不同的产业集群需求，设计不同的办公场地风格，同时提供家具、绿植、办公设备、装修、健康管理、保洁与饮用水、企业VI及网站设计等办公配套服务
2. **财务管理**：包括公司注册、代理记账、公司注销、公司变更和税务服务、工商服务、社保服务等
3. **办公装修**：专业的装修设计服务，为企业打造现代化、高效的办公空间，提升企业形象和工作效率
4. **营销策划**：为企业提供品牌策划、市场推广、数字营销等服务，帮助企业提升市场竞争力
5. **政策咨询**：专业政策咨询服务，帮助企业了解并享受政府优惠政策，降低运营成本

### 发展历程
- **2009年**：初创探索·奠定基础（公司创立初期，开始为企业提供工商注册、财务代理等基础服务，奠定企业发展基础）
- **2012年**：一期大厦1.25万平米投入使用（一期大厦1.25万平米正式投入使用，标志着公司正式进入写字楼运营管理领域）
- **2017年**：建设完成"龙溪商务中心"，形成了从孵化器助长成长、加速器茁壮成长到产业基地育树成林的生态链
- **2020年**：规模扩大·总量提升，二期项目开始投入使用，园区已初步形成了软件集成电路设计、电子商务、服务外包等产业集群
- **2025年**：拿下美峰创谷5号楼，3.5万平方写字楼酒店综合体运营
